etna-cli's documentation
========================

`etna-cli` is a CLI tool to interact with ETNA's intranet

The following command-line tools are included:

* ``etna config``:        Init, edit or delete etna config.

* ``etna conversation``:  Conversations on intranet.

* ``etna declare``:       Declaration

* ``etna event``:         Events.

* ``etna gitlab``:        Interact with Gitlab.

* ``etna project``:       Interact with Projects.

* ``etna rank``:          Get rank by promotion.

* ``etna student``:       Student stuff.

* ``etna task``:          Add quests and projects to TaskWarrior.

* ``etna ticket``:        Tickets stuff.
